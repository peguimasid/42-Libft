## Install

```sh
git clone https://github.com/peguimasid/42-Libft.git libft
```

## Usage

> Clone libft in you project directory

```c
#include "libft/libft.h"
#include <stdio.h>

int	main(void)
{
	char	*str = "My string";
	printf("%zu\n", ft_strlen(str));
}
```

## Run

```sh
cd libft && make && cd .. && gcc main.c ./libft/libft.a && ./a.out
```
